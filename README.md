# Alzheimer’s Disease Prediction

This project uses machine learning to predict the likelihood of Alzheimer’s disease
based on patient medical and cognitive data.

## Features
- Data preprocessing and analysis
- Machine learning classification model
- Model deployment using Python

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook

## Outcome
The model predicts whether a person is likely to have Alzheimer’s disease or not.
